## ⚡ What is Asok?

Asok is designed to combine the simplicity and performance of lightweight micro-frameworks with the features needed to build modern, interactive, and secure web applications.

* **Islands Architecture (SSR & Hydration)**: Server-side pre-rendering with selective HTML hydration and background stale-cache warming (ISR).
* **Native Reactive UI**: Stateful, real-time reactive components powered by WebSockets (no bulky SPA frameworks needed).
* **Modular Extensions**: A robust, secure plug-and-play extension system.
* **Modern Developer Experience**: Built-in developer toolbar, auto-docs, hot reloading, and native Vector Search.
